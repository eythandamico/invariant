# Divider

Horizontal rule using the theme border color.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| spacing | `'none'\|'sm'\|'md'\|'lg'` | `'md'` | Vertical margin |
| className | `string` | `''` | Additional CSS classes |

## Dependencies

- `tokens/theme.css`

## Usage

```jsx
<Divider />
<Divider spacing="lg" />
```

## CSS Variables Used

`--inv-border`
